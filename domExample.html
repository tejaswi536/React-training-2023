What is DOM?

The Document Object Model (DOM) in JavaScript is like a tree-like structure that represents a webpage. Each part of the page, like paragraphs, images, and buttons, is a "node" in this tree. You can use JavaScript to interact with and change these nodes. For example, you can add new content to a webpage, change the text in a paragraph, or make a button do something when clicked. The DOM helps JavaScript talk to and control the elements on a webpage.

Modifying HTML using the DOM (Document Object Model)

console.log(document) and console.log(window) are both logging objects, but they represent different things in the context of a web browser.

console.log(document): The document object represents the entire HTML document. It is part of the DOM (Document Object Model) and provides an interface to interact with the content and structure of the document. You can use document to access elements, modify content, update styles, and perform various operations on the document itself.

console.log(window): The window object is the global object in a browser environment. It represents the browser window or tab and encompasses the entire browsing context. window includes properties and methods that relate to the browser environment, such as handling events, opening new windows, and setting timeouts.
