Html basics

HTML(Hyper text markup languageHTML stands for HyperText Markup Language. It's the standard markup language used to create and structure web pages and their content. HTML provides a set of elements or tags that define the different parts of a web page, such as headings, paragraphs, images, links, lists, and more.

  • Markup Language: HTML uses tags to structure content, indicating to web browsers how to display text, images, links, and other elements.
    • Structure of Web Pages: It establishes the structure of a web document by defining elements like headings, paragraphs,lists, and more.
  • Creates structured content
  • Uses tags for formatting
  • Facilitates multimedia integration
Tag Description
html Defines the root of an HTML document
Head Contains meta-information about the document
Body Contains the visible content of the document
p Defines a paragraph
(h1 to h6 ) Headings
(p) Paragraphs
